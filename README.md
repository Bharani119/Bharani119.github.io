# Bharani119.github.io

## Overview

`Bharani119.github.io` is a lightweight personal portfolio website built as a single static page (`index.html`) with inline CSS. It’s intended for GitHub Pages and contains a hero section, an about/skills section, featured project cards, certifications, and contact links.

## Features

- Responsive layout (grid-based sections)
- Animated/gradient UI (hero intro + CTA buttons)
- Smooth in-page navigation (buttons scroll to `#projects` and `#contact`)
- Animated background and styled badges/cards

## Tech Stack

- HTML
- CSS (inline, no external dependencies)
- Small amount of inline JavaScript for smooth scrolling

## How to Run (Local)

1. Open `index.html` in your browser.
2. (Optional) Edit `index.html` and refresh the page.

## Sections

- `About Me`: background + core expertise + AWS services badges
- `Featured Projects`: 6 highlighted GenAI/agentic/RAG project cards
- `Certifications`: listed professional certifications
- `Let’s Build with AI`: email, GitHub, and LinkedIn links
